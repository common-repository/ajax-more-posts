Tags: ajax, loop, articles

Load infinite articles on the homepage by clicking a button

== Description ==

An AJAX Infinite scroll for articles on the homepage

Plugin developed by <a href="http://www.savingsaccountfinder.com.au">Savings Account Finder</a>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ajax-more-posts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Wrap `<div id="posts-list"> </div>` around your post loop
4. Create a loop.php file in your template directory with a copy of the same loop
5. Style as you wish
